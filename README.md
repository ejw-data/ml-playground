## Setup and Installation  
1. Environment needs the following:  
    *  Python 3.6+   
    *  scikit-learn
1. Activate your environment
1. Clone the repo to your local machine
1. Navigate the terminal to the repo folder
1. In the terminal, run `python main.py`
